40 Days of javaScript : DAY 13

JavaScript this — Normal vs Arrow Function

# Situation Function Type Value of this
1 Global Scope (non-strict) Normal window
2 Global Scope (strict) Normal undefined
3 Global Scope Arrow Inherits from outer scope (window)
4 Inside Object Method Normal Refers to the object itself
5 Inside Object Method Arrow Inherits from outer scope, not the object
6 Standalone Function (non-method) Normal window (non-strict) / undefined (strict)
7 Standalone Function (non-method) Arrow Inherits from enclosing lexical scope
8 Constructor Function Normal Refers to newly created object
9 Constructor Function Arrow this is inherited — likely undefined or global
10 Inside setTimeout / Event Handler Normal window or DOM element (depends on context)
11 Inside setTimeout / Event Handler Arrow Inherits from outer lexical scope
