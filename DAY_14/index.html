JavaScript Error Handling Keywords

Keyword Usage Example
try Wraps code that may throw an error. If error occurs, control jumps to catch. try { riskyCode(); }
catch Handles the error thrown in the try block. catch(err) { console.log(err); }
throw Used to manually throw a custom error or object. throw new Error("Something went wrong");
rethrow Throw the error again after partially handling it. catch(e) { if (!e.known) throw e; }
Error Object Provides information about the error (name, message, stack). err.name, err.message, err.stack
