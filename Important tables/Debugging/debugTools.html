Shortcut Name Description
F9 Continue (Step-by-Step) Executes the code line by line. Automatically steps into functions, executes them, and returns.
F10 Step Over Skips over function internals; executes them as a single step unless breakpoints exist inside.
F8 Resume Execution Skips to the next breakpoint, useful for jumping over blocks of code quickly.
F11 Step Into When stepping with F10, use this to enter a function and debug its internal lines.
Shift + F11 Step Out Exits the current function and returns to the calling line in the outer scope.
